# NEUROLING — Gesture · Voice · English Trainer

**NEUROLING** — это локальное веб-приложение, футуристический тренажёр английского языка, который объединяет жесты рук, голос и речь в одном неоновом HUD-интерфейсе.

Приложение открывает доступ к камере и микрофону, видит вас в зеркальном окне и через **MediaPipe Gesture Recognizer** (от Google, бесплатно, без ключей) распознаёт жесты ваших рук. Вы проговариваете английскую фразу по словам, и после каждого слова показываете нужный жест нужной рукой — система открывает следующее слово. Параллельно Web Speech распознаёт вашу речь и подсвечивает сказанные слова зелёным, а TTS озвучивает каждое новое слово.

В тренажёре 10 базовых фраз, каждая ≥7 слов, в прошедшем времени (was/were).

---

## Правило жестов

Фраза раскрывается по словам. На каждом шаге нужно показать жест, который зависит от номера слова:

- **Нечётное слово** (1, 3, 5…) → 👍 **палец вверх, ЛЕВАЯ рука**
- **Чётное слово** (2, 4, 6…) → ✊ **кулак, ПРАВАЯ рука**

Жест засчитывается только при удержании выше порога уверенности (~320 мс), чтобы исключить ложные срабатывания. Поскольку жесты на чётных и нечётных шагах разные, прогресс надёжен даже при неточном определении руки. Требование правильной руки (`strictHand`) можно выключить кнопкой, а кнопка **Swap** меняет маппинг рук на лету.

## Как запустить локально

1. Откройте терминал и запустите локальный сервер-мост (Node 18+, без зависимостей):

   ```bash
   cd /Users/artemiimiller/TEST3 && node server.js
   ```

   Сервер отдаёт приложение **и** проксирует озвучку в kie.ai (ключ берётся из `.env`).

2. Откройте в браузере **Chrome**:

   ```
   http://localhost:8000
   ```

3. По клику на старт-оверлей разрешите доступ к **камере и микрофону** (один общий запрос). На `localhost` камера разрешена без HTTPS.

> Можно запустить и через `python3 -m http.server 8000` — приложение заработает, но премиум-голос
> (kie.ai) будет недоступен, и озвучка автоматически уйдёт на браузерный `speechSynthesis`.

## Озвучка через kie.ai (мост)

Голос подключён через **kie.ai → ElevenLabs** (`elevenlabs/text-to-speech-multilingual-v2`), ключ из
`.env` (`KIE_AI_API_KEY`). Он **никогда не уходит в браузер** — клиент обращается только к локальному
`POST /api/tts`, а `server.js` сам ходит в kie.ai и возвращает mp3.

- **Кэш на диске** (`.tts-cache/`) — каждое слово генерируется один раз, потом отдаётся мгновенно и бесплатно.
- **Префетч** — при открытии фразы все её слова генерируются заранее, поэтому пословная озвучка идёт без задержки.
- **Цепочка фолбэков:** kie.ai → личный ключ ElevenLabs (если введён на старте) → браузерный голос.

Эндпоинты моста: `GET /api/health`, `POST /api/tts { text, voice?, speed? } → audio/mpeg`.

## Заметки

- **Интернет обязателен.** MediaPipe (модель, wasm и vision-библиотека) грузятся с CDN; озвучка идёт через kie.ai.
- **Ключ ElevenLabs опционален.** Премиум-голос уже работает через kie.ai. Поле на старте — для своего личного ключа ElevenLabs (необязательно).
- **Chrome рекомендован.** Распознавание речи использует `webkitSpeechRecognition`, который стабильно работает в Chrome. В браузерах без поддержки речь просто не подсвечивается (это дополнительный сигнал и не блокирует прохождение).

## Структура файлов

```
TEST3/
├── server.js           # Локальный сервер-мост: статика + прокси озвучки в kie.ai (+ кэш)
├── index.html          # Разметка HUD, точные id элементов, подключение js/main.js
├── styles.css          # Премиум тёмная тема (Clavion), анимации, метр уверенности
├── README.md           # Этот файл
├── .env                # KIE_AI_API_KEY (секрет, не отдаётся клиенту)
├── .tts-cache/         # Кэш сгенерированных mp3 (создаётся автоматически)
├── план/
│   └── PLAN.md         # Единый источник правды: контракт интерфейсов
└── js/
    ├── config.js       # CONFIG (mediapipe, gesture, kie, elevenLabs), GESTURE, expectedFor(step)
    ├── phrases.js      # 10 базовых фраз (прошедшее время)
    ├── camera.js       # startCamera(videoEl): запрос камеры + микрофона
    ├── gestures.js     # createGestureEngine: обёртка MediaPipe (VIDEO mode)
    ├── speech.js       # createSpeechMatcher: распознавание речи и матч слов
    ├── tts.js          # createTTS: озвучка (kie.ai → ElevenLabs → браузер) + prewarm
    ├── trainer.js      # createTrainer: конечный автомат тренажёра
    ├── ui.js           # createUI: рендер HUD по state
    └── main.js         # Бутстрап и проводка всех модулей
```
